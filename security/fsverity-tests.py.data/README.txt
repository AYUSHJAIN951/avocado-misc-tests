CONFIG_FS_VERITY: configuration parameter need to be set to continue with the
fsverity tests.

fsverity helps in authenticity protection for read-only files.
This test case runs 'make check'.

README for fsverity can be found at:
https://git.kernel.org/pub/scm/linux/kernel/git/ebiggers/fsverity-utils.git/tree/README.md

Kernel documentation can be found at:
https://www.kernel.org/doc/html/latest/filesystems/fsverity.html
